`loopstack create <dir>` scaffolds a new Loopstack app: NestJS boilerplate via `nest new`, `LoopstackModule.forRoot()` wiring, a zero-config hello workflow (deterministic — no API keys needed), docker-compose for Postgres/Redis/Studio, `.env.example`, and git init. `--skip-install` and `--no-git` flags supported.
